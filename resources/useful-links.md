## ‚ö†Ô∏è Common Pitfalls to Avoid

1. **Data Leakage** ‚Äî Never fit transformers on test data
2. **Overfitting** ‚Äî Always validate with held-out data
3. **Wrong Metrics** ‚Äî Use appropriate metrics for your problem type
4. **Forgetting Seeds** ‚Äî Set random_state for reproducibility
5. **Committing Data** ‚Äî Use .gitignore to exclude data files
6. **Last-Minute Work** ‚Äî Start early, commit often
